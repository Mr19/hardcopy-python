# Getting Started

## Installation

1. Install hardcopy via `pip`

  ```bash
  $ pip install -U hardcopy
  ```

2. Follow Steps 1 and 2 (here)[https://developers.google.com/drive/v3/web/quickstart/python]
  - for Unix-like systems, put the `client_secret.json` file in `~/.hardcopy/`
